# Quickstart Skill

A general-purpose analyst prompt for exploring your ACP data.

## Usage with Claude Desktop

1. Open Claude Desktop
2. Create a new Project
3. Paste the contents of `claude-project.md` into the Project system prompt
4. Start asking questions

## Usage with Other Agents

The prompt in `claude-project.md` is framework-agnostic. Use it as a system prompt with any MCP-compatible agent: CrewAI, LangGraph, Cursor, Claude Code, etc.
